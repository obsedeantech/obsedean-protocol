# Obsedean

> The AI-Native IDE for Solana.

Obsedean is an AI-native development environment built specifically for Solana builders. Powered by ARIA, a unified operator that controls wallets, token launches, Git workflows, deployments, and agent swarms entirely through natural language.

One operator. One workspace. Infinite execution. Inspired by the emerging trend of AI-native Solana workspaces that combine coding, wallets, deployments, and autonomous agents into a single environment. :contentReference[oaicite:0]{index=0}

---

## 🚀 Overview

Building on Solana often means juggling multiple tools:

- Code editor
- Wallet
- Terminal
- Git
- Deploy pipelines
- Launch tools
- AI assistants

Obsedean unifies everything into a single AI-native workspace driven by ARIA.

Instead of learning commands, builders simply describe what they want.

```text
"Launch a token."

"Deploy this Anchor program."

"Create a new feature branch."

"Audit this smart contract."

"Run 8 agents to refactor the codebase."
```

ARIA handles execution across the entire development lifecycle.
